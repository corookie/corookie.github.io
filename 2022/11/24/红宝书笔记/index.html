html中的javascript

​ 使用script标签把js代码插入html中，可以使用src属性引入外部代码，有了src则不执行行内代码，同时src不受同源策略的限制。script 标签一般放在body的尾部，这样浏览器加载更快。

​ script标签有很多属性

  • defer ： 可以让脚本在页面渲染解析完后再运行，只对外部脚本文件才有效，推迟的脚本在页面解析到 /html 再开始执行，但都在DOMContentLoaded之前执行
  • async：异步，让浏览器不必等脚本下载和执行完毕再渲染页面，因此异步脚本不应该修改dom

​ 同时还可以动态加载脚本，只需要在script标签中使用creatElement添加一个script标签,再给src赋值即可，这种添加的脚本会以异步形式被加载

除非使用XHTML，不然别指定script标签中type的属性

语言基础

var、let和const

​ var作用域为函数作用域，并且声明变量会提升到作用域的顶部，在全局中声明的变量会添加到windows对象上；

​ let作用域为块级作用域，出了块级会被销毁，let不能声明一个变量两次，并且不会在作用域置顶，很适合在循环内使用

​ const和let基本相同，但用const声明必须有初始值，并且const声明的变量修改会报错，因此很适合用来声明那些不变的量。

使用规则：const优先，用来声明那些常量，let次之，var不使用

数据类型

七大数据类型: null 、undefined 、 boolean 、number、string 、symbol ；object

简单数据类型

  • undefined 未定义，对未定义的变量只能使用type of，不要显式的给变量赋undefined
  • null 空指针 常用于初始化对象，给予对象一个初始值
  • boolean 布尔值 ,true or false ,可以通过某些方法转换为数值 0/1
  • number 包括整数和浮点数，2、8、16进制，NaN(NOT A NUMBER)包含NaN的操作始终返回NaN，number(),parseInt(),parseFloat()。对象转换为数值调用对象的value of
  • string 字符串，原始值，不能更改只能销毁重建，模板字符串，其他数据可以通过toString().方法转换为字符串，string.raw可以获取模板字符串里的原始字符
  • symbol 符号 ，每一个symbol()都是独一无二的，可用作对象的属性名。

复杂数据类型

  • object 对象，key：value，每个对象都有其构造函数

变量作用域和内存

原始值和引用值

引用值只有对象，也就是说接受对象赋值的变量都指向同一个对象，而原始值不同，赋值的时候会复制一份值，且改动不影响原变量

传递参数：按值传递(类似函数的参数)，按引用传递(有些对象的引用会按引用传递)

上下文和作用域

上下文：储存变量和和函数的地方(上下文中的关联对象上)

简单来说，内层的函数可以访问外层的变量，外层的无法访问内层的。内部上下文可以通过作用域链访问外部上下文中的一切，但外部上下文无法访问内部上下文中的任何东西。

垃圾回收

垃圾处理程序不是一直运行的，而是有间隔的；

回收方法主要是标记清理：变量再进入上下文和出上下文都会被标记，每次垃圾处理程序会标记内存中的所有变量，不过会把上下文或者以及上下文中引用的变量

的标记给去掉；这样的话那些身上带标记的变量就是待删除的变量；简单来说离开作用域的值都会被删除

隐藏类和删除操作

内存小，则性能就会很好，因此写代码时尽量减少内存的占用

数据无疑是内存的主要用户，对不用的数据最好的方式是解除引用，设置为null，这样可以减少内存

V8引擎有个隐藏类功能也可以减少内存的占用
